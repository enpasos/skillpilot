## Metrics Definitions
- **Atomic Skill**: A verifiable, binary competency (e.g., "Can differentiate Offer vs. Invitatio").
- **Cluster**: A grouping container (e.g., "Handelsrecht").
- **Depth**:
    - **Ultra-Deep (4)**: Examination Schemes (e.g., "TBM 1, 2, 3").
    - **Hyper-Deep (5)**: Definitional Breakdown (e.g., Defining "TBM 1" into component parts/problems).
- **Cluster**: Only modeled at the high-level handbook topics.
